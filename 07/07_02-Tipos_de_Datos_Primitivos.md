# 2. ¿Cuáles son algunos tipos de datos JS?

Todos los tipos de datos primitivios incluídos por defecto en JavaScript son:

* Booleanos
* Nulls
* Undefined
* String
* Numbers
    - Integers
    - Floats
    - Big Integers
* Symbols
